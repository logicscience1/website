Internet Radio Stream

It’s not always online but sometimes I like to play music and “DJ” so occasionally I will publish an Icecast stream on this link for anyone who might like my style of ‘jungletec’ mixing.

I use Mixxx DJ software when broadcasting as it connects easily with the Icecast server but eventually I’d like to be using the DJuced software native to the Hercules P32 mixer that I use. Unfortunately DJuced is not built for configuration with Icecast so I either need to wait for an update or work out an alternative that possibly uses podcasting software.



Soundcloud

Often I record my mixes for listening back, getting to know the tunes and whether the flow sounds any good. I upload them to my soundcloud profile which generally holds the current mix that I’m listening to.
